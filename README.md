# vue2-admin-element

> 这是一个极简的 vue2-admin 管理后台，基于vue-admin-template大佬项目的简化版。它只包含了基础必备功能：权限、数据报表、form表单、富文本、图片查看器等这些搭建后台必要的东西。

[演示地址](http://119.91.149.27:82/)
